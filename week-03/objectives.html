Effective Programming Practices for Economists, Winter Term 2020/21

Universität Bonn, Department of Economics

Prof. Dr. Hans-Martin von Gaudecker

Learning Objectives

Methods

  • Students appreciate the need for a robust version control system
  • Students appreciate the need for a robust collaboration tool
  • Students understand the DAG underlying Git
  • Students understand the relation between working copy, staging area, local repository, and central repository
  • Students retrieve earlier versions of a project
  • Students apply patterns of names
  • Students work with technical language

Tools

  • Students understand the basic usage of Git from the command line
    • git add
    • git commit
    • git clone
    • git pull
    • git push
    • git restore
    • git reset
    • git switch
    • git tag
  • Students analyse the revision history of a repository.
  • Students know how to ignore patterns of files in Git.

Personal

  • Students persevere when facing errors described in technical language
  • Students reflect on their own contribution to group work along the dimensions of responsibility, thinking abilities, cooperation, and autonomy & initiative

Social

  • Students discuss the most important dimensions of responsibility and thinking abilities
  • Students rate their peers’ contribution to group work along the dimensions of responsibility, thinking abilities, cooperation, and autonomy & initiative
  • Students make sure that group members who have to miss class time can catch up
