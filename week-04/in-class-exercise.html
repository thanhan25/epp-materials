Task 4: Re-write the example the from second video using partial (25min)

  • Partial evaluation allows you to pass some parameters to a function before passing the remaining parameters.
  • Specifically, you generate a new function based on another one.
  • Relative to the old function, the interface shrinks because some parameters are already evaluated.
  • Here is an abstract example:
from functools import partial


def f(x, y):
    return x + y


g = partial(f, y=7)

print(g(x=1))
  • Take the example from the second screencast.
  • Note that we doubled up the preference parameters purely to fit map()’s interface.
  • Use partial() to simplify the map() call so that it takes only two arguments, not four.
