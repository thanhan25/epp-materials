Material 3: Run and write simple tests yourself

In this very directory, you will find the module utility_crra.py, which is the same as last week. You also find utility_crra_test.py, which is the same as the initial version in the second screencast.

Copy the file over Four little tasks:

  1. Run pytest.

  2. Adjust utility_crra.py so that the test passes.

  3. Implement the test for log utility as in the screencast.

  4. Implement two further tests for values of argument, which are:

    • equal to zero;
    • smaller than zero.

This should be very fast. The sole purpose is that you are broadly acquainted with it and do not spend precious class time getting things going.

Do not spend more than 15 minutes on this.
